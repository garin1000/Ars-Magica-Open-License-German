german-ordered – Alphabetisch sortierte Übersetzungen

Automatisch erzeugte Versionen der freigegebenen Übersetzungen aus /german-reviewed mit alphabetisch sortierten Einträgen (Tugenden, Fehler, Fertigkeiten, Zauber, Indizes, Qualitäten).

Erzeugt durch /sort-entries (tools/sort-entries/sort_entries.py). Dient als Eingabe für /build-html.
