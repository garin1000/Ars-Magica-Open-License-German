# german-html – HTML-Regelwerke

Self-contained HTML-Dateien der Regelwerke, erzeugt aus [`/german-ordered`](../german-ordered/) durch `/build-html` (`tools/build-html/md_to_html.py`).

Enthalten einklappbare Navigation (links, H1–H3), Fuzzy-Suche mit Fuse.js (rechts) und bildschirmoptimiertes CSS. Keine externen Abhängigkeiten — alles inline in einer Datei pro Regelwerk.
