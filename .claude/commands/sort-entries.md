# Einträge der Basisregeln alphabetisch sortieren

Sortiere alle relevanten Abschnitte (Tugenden, Fehler, Fertigkeiten, Zauber, Indizes) der deutschen Basisregeln alphabetisch und lege das Ergebnis in `german-ordered/` ab.

---

## Schritt 1 — Skript ausführen

Führe das bestehende Skript aus:

```bash
python3 tools/sort-entries/sort_entries.py
```

## Schritt 2 — Ergebnis prüfen

1. Prüfe, dass die Zeilenzahl von Ein- und Ausgabe identisch ist (steht im Sortierbericht).
2. Gib dem Benutzer den Sortierbericht aus.
3. Falls die Zeilenzahl abweicht, melde dies als Fehler.
