## Statblock-Begriffe / Creature Stat Block Terminology

| Englisch (EN) | Deutsch (DE) | Anmerkung |
|---|---|---|
| Creature | Kreatur | Oberbegriff für Tiere, Geister, Dämonen, Feen etc. |
| Animal | Tier | |
| Beast | Tier / Bestie | |
| Magical animal | Magisches Tier | |
| Cunning | Gerissenheit | Ersetzt Intelligenz bei Tieren; kann nicht Lesen/Schreiben |
| Intelligence | Intelligenz | Nur bei Wesen, die denken und lernen können |
| Size | Größe | Skala: −15 (Insekt) bis +10 (sehr großes Wesen); ±0 = erwachsener Mensch |
| Quality | Eigenschaft (Tier) | Positive tier­spezifische Eigenschaft (entspricht Tugend) |
| Inferiority | Schwäche (Tier) | Negative tier­spezifische Eigenschaft (entspricht Fehler) |
| Natural weapon | Natürliche Waffe | Klauen, Biss, Hufe, Hörner usw. |
| Vis (in creature) | Vis (in Kreatur) | Im Körper toter Wesen mit Machtwert; nicht im lebenden Körper |
| Species virtue / flaw | Arteigene Tugend / Fehler | Kostenlos; typisch für die gesamte Art |
| Penetration | Penetration | Kreaturenkraft-Penetration: Machtwert − (5 × Machtpunktkosten) + Bonus |

## Größentabelle / Size Table

| Größe | Beispiele |
|---|---|
| −5 bis −4 | Kleine Raubtiere, Vögel |
| −3 bis −2 | Dachs, Katze, Fuchs, kleines Kind |
| −1 bis 0 | Jugendlicher, Wolf, Hirsch, Erwachsener Mensch, Schwein |
| +1 | Kapitalhirsch, Pony |
| +2 | Bär, Pferd, Löwe |
| +3 | Auerochse, Elch, Schlachtross |
| +4 | Elefant |
| +5 | Killerwal |
| +7 und mehr | Drachen, Wale |

**Größenänderung:** Pro Größenpunkt Reduktion: −2 Stärke, +1 Schnelligkeit, −1 Wundgrenze. Umgekehrt bei Vergrößerung.
