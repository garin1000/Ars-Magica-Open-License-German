# Ars Magica Definitive Edition – Übersetzungsregeln und Formatierung

**Quelle:** ArM_DE_Uebersetzungstabelle.md (Sek. 23–24)

---

## Übergeordnete Regeln zu Übersetzung

### Deutsche Sätze

Verwende **natürliche** deutsche Satzstrukturen. Wenn der Text im Englischen formal ist, soll er auch im Deutschen formal sein. Übernimm den Satzbau nicht direkt aus dem Englischen.

### Semikolons

Verwende Semikolons nur, wenn sie notwendig sind.

---

## Allgemeine Regeln zur Formatierung

### Prozentzeichen

Vor einem **Prozentzeichen, das zu einer Zahl gehört**, steht nie ein Leerzeichen:
```
9%
9,9%
Mit % markiert.
```

### Tabellenformatierung

Tabellen sind **immer** so zu formatieren, dass alle Spalten außer der letzten in der Quelltextansicht ausgerichtet sind:
```
| Spalte 1 | Spalte 2         | Spalte 3 |
| -------- | ---------------- | -------- |
| 0        | Das ist ein Text | Dieses ist eine längere Erklärung |
| 1        | Kurz             | |
```

### Abkürzung n/a

Die englische Abkürzung **n/a** (not applicable / not available) wird mit **n/v** übersetzt.

### Ausprachehilfen

Die für englischsprachige Leser gedachten Aussprachebeispiele [z.B. (DAH-van ALL-ath)] werden weggelassen. Um die Zeilensynchronität zu erhalten, wird die weggelassene Zeile durch eine Leerzeile in der deutschen Datei ersetzt (oder der verbleibende Text so auf die Zeilen verteilt, dass die Gesamtzahl gleich bleibt).

---

## Listen und Aufzählen

Verwende Listen und Aufzählungen **nur** wenn sie auch im Original so formatiert sind. Ändere **niemals Fließtext** zu Listen oder Aufzählungen.
