# lektorat – Lektoratsergebnisse der Prüfagenten

Prüfagenten legen ihre Ergebnisse nach jeder Übersetzungsprüfung hier ab. Lektorat und Übersetzung werden von **unabhängigen Agenten** durchgeführt.

---

## Dateinamenskonvention

Dateiname analog zur geprüften Quelldatei, z. B.:
- `lektorat/core-rules-kap3.md` für Kapitel 3 der Core Rules
- `lektorat/ArsMagica_DE_Gesamt-kap7.md` für einen Abschnitt aus der Gesamtübersetzung

---

## Inhalt einer Lektoratsdatei

Jede Lektoratsdatei enthält:
- **Gefundene Fehler** – mit Zeilenverweis auf die geprüfte Datei
- **Stilkorrekturen** – zu direkt aus dem Englischen übernommene Konstruktionen → natürlicheres Deutsch
- **Offene Fragen** – Terminologie-Unsicherheiten, Interpretationsspielräume
